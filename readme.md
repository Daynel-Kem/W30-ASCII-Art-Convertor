A simple python script that takes an image and converts it into ASCII art  
  
to use it  
`python ascii.py image_path {height} {width}`  
where `{height}` and `{width}` are integer sizes of how large you want each pixel to be 

Example:  
![My Logo to be Converted into ASCII](./input_images/output-onlinepngtools.png)  
  
Using the command
`python ascii.py /input_images/output-onlinepngtools.png 12 8`
